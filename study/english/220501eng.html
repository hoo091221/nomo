현재 진행형

쓰임 - 현재 진행 중인 동작을 나타낼 때 쓰며, '~하는 중이다', '~하고 있다'의 의미로 해석된다.
형태 - (주어) + (be동사의 현재형 am/are/is) + 동사의 -ing 형태 ~.

동사 만드는 법 예
대부분의 동사 + -e read => reading
work => working
walk => walking
talk => talking
-e로 끝나는 동사 e를 제거하고 + -ing dance => dancing
have => having
shake => shaking
make => making
'자음 + 모음 + 자음'으로 끝나는 동사 마지막 자음을 한번 더 쓰고 + -ing swim => swimming
cut => cutting
shop => shopping
run => running
'자음 + y'로 끝나는 동사 마지막이 '자음 + 모음 + 자음'으로 끝나도 그냥 + -ing 만 붙인다. play => playing
fly => flying
study => studying
enjoy => enjoying
'모음 + 모음'으로 끝날 때(보통 -ie로 끝난다) 마지막 '모음 + 모음'중 첫 번째 모음을 'y'로 고치고, 두 번째 e를 삭제한다. die => dying
lie => lying

현재진행형의 긍정문

형태 - 주어 + be동사의 현재형 + not + 동사의 ing 형태~.

현재진행형의 부정문

형태 - 주어 + be동사의 현재형 + not + 동사의 ing 형태~.
축약 - be동사와 not를 축약할 수 있다.

현재진행형의 의문문

형태 - be동사 + 주어 + 동사의 ing 형태 ~?
대답 - Yes 또는 No로 대답한다.
(Yes (주어) (be동사).)

Will

~할 것이다 라는 미래의 의미

현재진행형의 긍정문

형태 - 주어 + will + 동사 원형

현재진행형의 부정문

형태 - 주어 + will + not + 동사 원형 축약 - 주어 + won't + 동사 원형 [won't는 운-트 라고 읽는다]

현재진행형의 의문문

형태 - Will + 주어 + 동사 원형 ~? 대답 - Yes 또는 No로 대답한다.
(Yes (주어) will.)
