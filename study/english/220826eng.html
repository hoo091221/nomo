과거 시제

의미

현재의 시점이 아닌, 과거의 시점을 나타낼 때 사용한다.

형태

be동사의 과거형 : am/is -> was, are -> were
일반동사의 과거형

대부분의 동사 -e로 끝나는 동사 '자음 + y'로 끝나는 동사 '단모음 + 단자음' 으로 끝나는 동사 불규칙 동사
+-ed + -d -y -> -ied 마지막 자음 추가 + -ed have -> had go -> went come -> came
say -> said do -> did buy -> bought
make -> made feel -> felt bring -> brought
work -> worked want -> wanted like -> liked close -> closed cry -> cried study -> studyied stop -> stopped drop -> dropped

부정문과 의문문

구분 be동사 일반동사
부정문 주어 + was(were) not ~.
 주어 + did not + 동사원형 ~.
의문문 Was(Were) + 주어 ~? Did + 주어 + 동사원형 ~?

예문

I was late for school yesterday.(나는 어제 학교에 지각했다)
I played basketball with my friend last Friday.(나는 지난 금요일에 친구들과 농구를 했다)
