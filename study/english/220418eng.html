일반 동사

주어의 동작이나 상태를 나타내는 말(~하다)
형태 - 주어가 3인칭 단수일 경우(he, she, it 등) 일반동사의 현재형은 동사원형 뒤에 -(e)s를 붙인다.

종류 만드는 법 예
대부분의 동사 + -s cut => cuts
cook => cooks
eat => eats
play => plays
-o, -s, -x, -ch, -sh로 끝나는 동사 + -es do => does
pass => passes
go => goes
teach => teaches
'자음 + y'로 끝나는 동사 y를 i로 고치고 + -es cry => cries
fly => flies
study => studies
try => tries
불규칙동사 모두 다르다(외워야 한다) have => has

일반동사의 부정문

형태 - 주어 + do(주어가 3인칭 단수일때는 does) + not + 동사원형~.
축약 - do not => don't, does not => doesn't

일반동사의 의문문

형태 - Do/Does + 주어 ~?
대답 - Yes 또는 No로 대답한다.
(Yes 주어 do(does).)
