세계의 여러 문화 지역

용어 정리

문화 : 인간과 환경이 상호 작용하는 과정에서 형성된 공통된 생활 양식
문화 지역 : 같은 문화 요소릴 공유하거나 비슷한 문화 경관이 공통적으로 나타나는 공간적 범위(언어, 종교 등의 기준에 따라 달라질 수 있음)

동아시아 문화 지역

속해있는 나라 : 대한민국, 일본, 중국 등....
종교 : 예로부터 불교가 발달하였다.
언어 : 한자를 주로 사용한다.(한국어, 중국어, 일본어 모두 기반은 한자)
기타 : 유교 사상, 벼농사 발달, 젓가락을 사용한다.

동남아시아 문화 지역

속해있는 나라 : 베트남, 필리핀, 태국 등....
종교 : 종교가 매우 다양하다.
언어 : 매우 다양한 언어를 사용한다.
기타 : 인종이 다양하고, 벼농사 발달, 중국과 인도의 영향 많이 받음

인도 문화 지역

속해있는 나라 : 파키스탄, 방글라데시, 스리랑카....
종교 : 인구가 많아 종교가 다양하다. 힌두교와 불교의 발상지이다.
언어 : 같은 나라 사람들끼리도 말이 안 통할 정도로 매우 다양하다.
기타 : 카스트 제도(인도만의 신분 제도)의 영향이 남아 있다.(법적으로는 사라짐)

+Plus
카스트 제도
인도의 신분 제도이며, 높은 계급부터 브라만(귀족), 크샤트리아(무사), 바이샤(농민, 상인), 수드라(노예)이다.

건조 문화 지역

속해있는 나라 : 사우디아라비아, 알제리, 기니 등....
종교 : 주민 대부분이 이슬람교를 믿는다.(알라!)
언어 :
기타 : 건조 기후, 아프리카 북부 지역과 서남아시아 지역이 같이 있고, 유목과 오아시스 농법 발달.

아프리카 문화 지역

속해있는 나라 : 남아프리카 공화국, 모잠비크, 콩고 민주 공화국 등....
종교 : 이슬람교나 기독교를 믿거나, 전통적인 종교를 믿는다.
언어 : 유럽의 식민지배 영향으로 유럽 쪽 언어를 공용어로 사용
기타 : 사하라사막 이남 아프리카에 해당, 유럽의 식민 지배를 받은 국가가 많음, 부족 단위의 공동체 문화가 남아 있다.

유럽 문화 지역

속해있는 나라 : 영국, 프랑스, 독일 등....
종교 : 크리스트교 문화가 발달하였다.
언어 : 라틴어를 기반으로 한 언어(영어, 독일어 등)를 사용한다
기타 : 주로 백인들이 거주, 산업혁명으로 일찍 산업화

앵글로아메리카 문화 지역

속해있는 나라 : 미국, 캐나다
종교 : 크리스트교(개신교) 문화가 발달하였다.
언어 : 대부분 영어를 사용한다
기타 : 경제수준이 높고, 인종 구성이 다양하다. 초기에 앵글로족이라는 부족이 살아 앵글로아메리카라는 이름이 붙었다.

유럽 문화 지역

속해있는 나라 : 에스파냐, 포르투갈, 브라질 등....
종교 : 크리스트교(가톨릭교) 문화가 발달하였다.
언어 : 에스파냐어와 포르투갈어를 사용한다.
기타 : 원주민, 백인, 흑인, 혼혈족의 문화 형성

오세아니아 문화 지역

속해있는 나라 : 호주, 파푸아뉴기니 등....
종교 : 그리스도교 문화가 발달하였다.
언어 : 말레이어/폴리네시아어 사용
기타 : 유럽 문화의 영향을 많이 받음, 원주민이 많이 거주

북극 문화 지역

속해있는 나라 : 러시아, 캐나다 등....
종교 : -
언어 : 이누이트어
기타 : 순록 유목 등 추운 기후에 적응한 독특한 생활 양식

자연환경에 따른 문화 차이(의)

열대 기후 : 통풍이 잘 되는 앏은 옷
건조 기후 : 사막이 많아 강한 햇볕과 모래바람을 막아주는 온몸을 감싸는 긴 옷
한대 기후 : 추위로부터 몸을 보호하는 두꺼운 옷

자연환경에 따른 문화 차이(식)

계절풍 기후(온대기후) : 쌀을 주식으로 삼음
서늘하고 건조한 기후 : 밀을 주식으로 삼음
고산 기후 : 감자/옥수수를 주식으로 삼음

자연환경에 따른 문화 차이(주)

주변에서 쉽게 구할 수 있는 재료로 제작
열대 기후 : 열대우림에 나무가 많음. 비가 많이 오므로 지붕의 경사를 깊게 함.
건조 기후 : 나무가 없어 흙으로 집을 만듬. 유목을 할 경우 이동식 가옥을 지음.
냉대 기후 : 통나무로 집을 지음.
한대 기후 : 과거에는 얼음집(이글루)를 지었음. (현재는 그냥 서양식 주택)
