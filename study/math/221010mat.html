자료의 정리 1

줄기와 잎 그림

줄기와 잎 그림은 자료의 분포를 확인할 때 쓴다.
자료의 큰 자리 수와 작은 자리 수를 각각 줄기, 잎으로 가운데에 세로선을 두고 정리해 표로 나타낸 것이다.

줄기와 잎 그림을 그리는 방법

1. 우선 큰 자리 수와 작은 자리 수로 구분선을 긋는다.

2. 큰 자리가 같은 변량들을 한 줄에 순서대로 나열한다.

3. (1|0 = 10) 처럼 이 수의 큰 자리는 무슨 자리의 수인지, 작은 자리는 무슨 자리의 수인지를 설명하기
(위 그림에선 큰 자리가 십의 자리이고 작은 자리가 일의 자리임)

주의점

- 작은 자리를 쓸 때는 칸을 맞추어서 써 주어야 한다.
- (1|0 = 10), (1|3 = 130) 등을 표시해 주지 않으면 큰 자리 수가 십의 자리, 작은 자리 수가 일의 자리인 걸로 간주한다.
- 줄기는 중복된 수가 있을 시 생략이 가능하지만, 잎은 불가능하다.

장/단점

장점 : 자료의 분포 상태를 한눈에 알 수 있다.
단점 : 자료의 수가 많아지거나 단위가 커지면 정리하기 힘들어진다.

도수분포표

도수분포표는 자료의 분포 상태를 확인할 때 쓴다.

도수분포표의 개념과 기초지식

변량 : 자료를 수량으로 나타낸 것
계급 : 변량을 일정한 간격으로 나눈 구간
계급의 크기 : 변량을 나눈 구간의 너비
도수 : 각 계급에 속하는 자료의 수
도수분포표 : 주어진 자료를 몇 개의 구간으로 나누고 각 계급에 속하는 도수를 조사하여 나타낸 표

도수분포표의 개념과 기초지식

계급의 크기 : 20(분)

1. 주어진 자료에서 가장 큰 변량과 가장 작은 변량을 찾는다.

2. 가장 큰 변량과 가장 작은 변량이 모두 포함될 수 있도록 계급을 잡는다
이때, 계급의 크기는 모두 같게 하고, '00이상 00미만' 의 형태가 되게 한다.

3. 각 계급에 속하는 변량의 개수를 세어 계급의 도수를 정한다.

주의점

- '00이상 00미만'은 첫 번째 계급에만 쓴다.
- 각 계급의 도수의 합이 변량의 개수와 맞는지 본다.

장/단점

장점: 전체적인 경향이나 자료의 분포를 알기 편하다.
단점: 정확한 변량이 얼마인지 하나하나 알 수 없다.
