요약

요약이란?

- 독자가 자신의 방식으로 글의 핵심적인 내용을 정리하는 것
- 독자는 자신의 고유한 시각과 방법을 사용하여 중요하다고 판단되는 내용을 중심으로 정리
- 단순하게 글의 줄거리를 간추리는 것이 아니라, 독자가 글 속에 담긴 정보를 중요도에 따라 선택하여 구조화

요약하기의 필요성

- 글의 내용을 정확하게 이해할 수 있다
- 글의 내용을 체계적으로 파악하여 오래도록 기억할 수 있다
- 글에서 핵심 내용과 부수적인 내용을 구분하는 능력을 가질 수 있다

문단의 중심 내용을 요약하는 방법
1. 선택 - 글의 중심 내용이 직접 드러난 문장을 고르는 것
2. 삭제 - 중심 내용과 거리가 먼 부분을 지우는 것
3. 일반화 - 개별적, 구체적 내용은 이를 포괄하는 상위 개념으로 표현하는 것
일반화의 예시 - 익명이는 사과와 포도를 좋아한다 -> 익명이는 과일을 좋아한다
4. 재구성 - 중심 문장이 분명하게 드러나지 않을 때에 각 문장에 제시된 내용을 바탕으로 중심 문장을 새로 만드는 것
글의 종류 방법 과정
설명하는 글 설명 대상에 대한 주요정보를 요약 1. 글의 구조를 파악하며 읽기
2. 문단의 중심 내용 간추기리
3. 글의 구조를 고려하여 문단의 중심 내용 정리하기
4. 정리한 내용을 자연스럽게 연결하기
주장하는 글 글쓴이의 주장과 근거를 중심으로 요약
이야기 글 이야기의 구성 요소인 인물, 사건, 배경등을 중심으로 요약 1. 이야기의 흐름에 따라 중심 사건 정리하기
2. 중심 사건을 자연스럽게 연결하여 줄거리 요약하기
