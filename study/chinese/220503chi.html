한자어의 짜임

주어의 특징
- 맨 앞에 온다
- 명사/대명사/수사 + 은/는/이/가로 구성되어 있다

서술어의 특징
- 맨 뒤에 온다
- 주어의 동작(동사)이나 형태(형용사)를 나타내는 말이 온다.

주술 구조
주어+서술어의 형태로 되어 있고, 주어는 서술어이다의 형태가 된다

日出
날 일 나올 출 => 해가 뜨다

수식 구조
앞의 말이 뒤의 말을 꾸며주는 관계. 이때 앞의 말을 수식어, 뒤의 말을 피수식어라고 한다. 수식어는 형용사, 명사, 동사로 이루어져 있고, 피수식어는 명사만 될 수 있다.

善行
착할 선 행동 행 => 착한 행동
校長
학교 교 긴 장 => 학교의 우두머리

병렬 구조
두 개 이상의 한자를 나란히 줄세우는 구조

1. 유사 관계
A와 B의 뜻이 비슷한 한자를 나열한 구조

海洋
바다 해 바다 양
山岳
뫼 산 큰산 악

2. 상반 관계
A와 B의 뜻이 반대인 한자를 나열한 구조

父母
아버지 부 어머니 모 => 아버지와 어머니 上下
위 상 아래 하 => 위와 아래

3. 대등 관계
A 아니면 B, A 그리고 B의 의미를 나타내는 구조

春夏秋冬
봄 춘 여름 하 가을 추 겨울 동 => 봄 그리고 여름 그리고 가을 그리고 겨울

술목관계
'서술어 + 목적어'의 형태로 되어 있고, '동사/형용사 + 명사/대명사/수사'의 형태

讀書
읽을 독 책 서 => 책을 읽다

보어
목적어와 다르게 명사+은/는/이/가의 구조이고, 서술어의 설명에서 부족한 점을 보충해 주는 문장성분이다.

술보관계
'서술어 + 보어'의 형태로 되어 있고, 주어는 생략된다. 따라서 '보어가 서술어하다'의 형태이지만 서술어가 '아니다/맞다/있다/없다'가 올 때가 많다.

入學
들 입 학교 교 => 학교에 입학하다
有罪
있을 유 잘못 죄 => 죄가 있다

+Plus
팁 - 술목관계처럼 보이지만 목적어라 하기에 애메한 것들이 붙어 있으면 주로 술보관계이다.
